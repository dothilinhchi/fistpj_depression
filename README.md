# fistpj_depression
# Multimodal Depression Detection using AVEC2017 Dataset
### Overview:
This is my first research project focusing on Machine Learning applications for mental health. The goal is to detect depression signs by analyzing Audio, Text, and Video features from the AVEC2017 dataset.
### Current Status:
The model is currently trained and evaluated on a limited scope of 11 subsets.

Note: Due to the small sample size, the current results exhibit bias and the model's performance is not yet optimal.
### Future Works:

Scale up the training process to the full AVEC2017 dataset.

Fine-tune model hyperparameters and explore advanced multimodal fusion techniques to improve accuracy.
